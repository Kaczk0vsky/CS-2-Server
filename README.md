1. Download Docker Desktop: https://www.docker.com/
2. Repo - download and paste everything: https://github.com/LiammCodes/cs2-dedicated-server
3. Create Token: https://steamcommunity.com/dev/managegameservers
4. Paste the token into docker-compose.yml line 7. Change whatever is needed to change like name or add passwords.
5. Open Powershell console in the folder with server and write: "docker compose up".
